# coc-discord-presence

Update your Discord status based on your vim project

## Install

`:CocInstall coc-discord-presence`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-discord-presence-keymap)`

## Lists

`:CocList demo_list`
